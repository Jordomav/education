MAMP

MAMP is a great way for us to quickly set up a development server on everyone's machines no matter the operating system. MAMP is a free resource, you can get MAMP Pro if you would like but it is not required. MAMP has PHP and MySql already installed and connected. Everything comes preconfigured out of the box and all we have to do is tell MAMP where to look and make a few quick tweaks and we'll be ready to go.

What is CodeIgniter?

Make PHP more tolerable.

If you've ever worked with vanilla PHP before you know how much of a pain. CodeIgniter allows us quickly generate elegant web applications with ease. Suffice it to say, CodeIgniter makes building applications with PHP much more enjoyable.

A simple framework that leaves a small footprint.

Unlike another exceedingly popular, and indeed fantastic framework, Laravel; CodeIgniter is very minimalistic. It attempts to keep its hands off and only take control when necessary.

Work with MySql much easier.

CodeIgniter has a lot of database functions. This allows us to connect to databases and tables with ease. It also allows us to add, update, delete and retrieve data quickly and easily.

Writing our first bit of code...

Headers, like their name implies, are meant to act as titles. They vary from 1 to 5, one being the biggest and five being the smallest. We want to use these sparingly, however, and only a few should be on each page.

Other tags that you will commonly see are p, or paragraph tags. These are used to display most information, and we can use CSS (as we will do in the next part) to resize the tags. There are also a, or anchor tags. These are used for linking to other pages, whether that be a page on your site or another website all together. Another common one that we will be using is ul, or unordered list and li. We can use this to display lists of information, and we will be using this to create our navigation.

The HTML structure

  • <!doctype html> - defines to the webpage, as the name implies, what type of file is.
  • <head> - contains all information that our webpage will need to know before loading it and rendering it into the view.
  • <meta> - gives all information about other data that the page may need.
  • <body> - Is where the bulk of our code will go. Everything that you see on a website is contained inside this tag.

CSS Syntax

  • body - Is the selector, this is how we tell the CSS what element we want to apply our styling to.
  • background-color: red; - Is the declaration, this is where the styling that we want use goes.
  • background-color - Is the property, this is the different CSS "properties" that are available to us. You can look at more here.
  • red - Is the value, this is what we want to apply to the property that is mentioned before it.
  • : separates the property and value. ; separates each of our property and value pairs.
